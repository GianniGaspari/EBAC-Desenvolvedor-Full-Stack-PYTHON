Aula 01

Introdução + Stage

  • Adicionando Powershell na máquina, para usar o terminal
    • Comandos no terminal:
      • dir == listar o conteúdo da pasta
      • ls == listar o conteúdo da pasta
      • "control" + l == limpar a tela do terminal
      • cls == limpar a tela do terminal
      • "seta" para baixo ou cima == histórico de comandos no terminal
      • cd.. == para sair da pasta
      • cd == para entrar na pasta
      • mkdir == para criar uma pasta nova
      • .\ == depois de cd, para adicionar a pasta desejada
      • echo "" > arquivo.txt == para criar um arquivo de texto dentro da pasta
      • cat arquivo.txt == para ver o conteúdo criado no comando anterior
      • rm -r .\arquivo.txt == para deletar o arquivo
      • git config --global usar.name "Gianni Gaspari" == para adicionar nome no git
      • git init . == cria um repositório git
      • git status == para ver o estatus do repositório
      • git add == da um track no arquivo fora da pasta
      • git rm --cached .\arquivo.txt == tirar arquivo da pasta
      • git add . == adiciona todos arquivos fora de uma vez para dentro
